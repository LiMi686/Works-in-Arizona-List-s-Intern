# Lapsed Donor Reactivation

This subproject is intentionally separate from the `Activist Codes / Tags` governance work.

## Files

- `lapsed_donor_reactivation_notebook.ipynb`
  Primary Jupyter Notebook for defining, scoring, and exporting consistent-but-lapsed donors.
- `lapsed_donor_reactivation_report.md`
  Summary report generated by the notebook.
- `fundraising_outreach_executive_summary.md`
  Executive-style summary report with figures covering four outreach questions and recommended actions.
- `fundraising_outreach_executive_summary.ipynb`
  Jupyter Notebook version of the executive summary, organized as markdown and code cells for presentation and reruns.
- `figures/`
  Generated chart assets used by the markdown reports, including direct-mail opportunity, postcard opportunity, donor vs Leadership Council, and lapsed-donor priority visuals.
- `lapsed_consistent_donors.csv`
  Full candidate list.
- `lapsed_consistent_donors_250plus.csv`
  Priority segment with at least one `$250+` giving year.
- `people_added_since_january_postcard_outreach.csv`
  People added since January 1, 2026 who have a complete mailing address, no email on file, and are not flagged `nomail`.
