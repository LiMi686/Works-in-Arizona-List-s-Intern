**16,440 人**是一个重要的触达缺口：这批人无法通过电子邮件联系，但拥有可用的邮寄地址，且未设置邮件退订。对于无法被数字化渠道覆盖的潜在捐款人和关注者，定向直邮（明信片、信件）是目前唯一的主动触达方式。

## 五、流失捐款人识别与优先级分层

### 筛选标准

- **一致性捐款人：** 累计捐款年数 ≥ 3 年，且最长连续捐款年数 ≥ 3 年
- **流失定义：** 最后一笔捐款距分析日期在 **1 至 5 年之间**（即已停止捐款但未完全淡出）
- **高价值定义：** 至少有一年年度捐款金额 ≥ $250

### 连续性检测方法（Gaps-and-Islands）

利用 SQL 窗口函数计算每位捐款人的最长连续捐款年份序列：

```
gift_year − ROW_NUMBER() OVER (PARTITION BY vanid ORDER BY gift_year)
```

该差值在任何不间断的连续年份区间内保持恒定，从而识别连续捐款"岛屿"。
